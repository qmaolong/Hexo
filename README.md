## 操作说明

### 1、安装NodeJS

……

切换npm源到淘宝

`npm config set registry https://registry.npm.taobao.org`

### 2、安装Hexo

`npm install hexo-cli -g`

### 3、clone代码

`git clone https://github.com/qmaolong/hexo.git`

### 4、进入文件夹，根据package.json安装所需的ode组件

`npm install`

### 5、生成，部署
```
hexo g
hexo d
```
