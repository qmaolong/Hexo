1、安装nodejs
……

2、安装hexo：
npm install hexo-cli -g

3、之后，已经不需要初始化了

4、直接在任意文件夹下恢复文件：
git clone https://github.com/qmaolong/hexo.git

5、然后进入文件夹，安装node组件：
cd xxx.github.io
npm install
npm install hexo-deployer-git --save

6、安装渲染器（详见：https://www.haomwei.com/technology/maupassant-hexo.html）：
npm install hexo-renderer-pug --save
npm install hexo-renderer-sass --save

7、生成，部署：

hexo g
hexo d

8、然后就可以开始写你的新博客了：

hexo new newpage
